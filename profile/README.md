# HizenLabs

Tools and plugins for [Rust](https://rust.facepunch.com/) servers, built on
[Carbon](https://carbonmod.gg/) with [Oxide](https://umod.org/) compatibility.

We focus on clean, performant, well-documented mods — shared, reusable building
blocks for localization, UI, configuration, and pooling, packaged into
single-file plugins that drop straight onto your server.

## On AI

We use AI throughout this project, and we're open about it.

AI-assisted contributions are welcome — we won't reject a change because a model
helped write it. The requirement is the same one we hold all code to: a human has
read, understood, and reviewed every line before it lands, and can explain and
stand behind it. AI speeds the work; it doesn't replace review. Unreviewed or
unexplainable code doesn't merge, however it was produced.
